Como crear una página personal en GitHub

¡Hola a todos!

Tras mucho pensar que podía compartir no se me ocurre nada mejor que contaros qué debéis hacer para crearos vuestra propia página personal en GitHub. Así que, ¿qué os parece si comenzamos?.

En primer lugar debéis tener una cuenta en GitHub, los que no las tengáis solo tenéis que registraros en https://github.com/.

Una vez estáis en vuestro perfil es necesario que creéis un repositorio nuevo.

Una vez que estáis creandolo tendréis que poner como nombre del repositorio: tuusuario.github.io

Una vez creado nos vamos a Settings, buscamos la entrada que pone GitHub pages y pinchamos en Automatic page generator.

Con estos simples pasos ya tenemos creada nuestra página en GitHub.

Al pinchar en Automatic page generator os enviará a un editor para agregar el contenido que deseemos a nuestra página y acto seguido procederéis a la elección del tema para vuestra página.

Jekyll

Jekyll es un generador simple de blog que podéis utilizar en vuestra página de GitHub.

Para saber como instalar y utilizar Jekyll + GitHub os dejo los siguientes enlaces de interés para echaros una mano:

https://help.github.com/articles/using-jekyll-with-pages/

http://jekyllrb.com/docs/installation/

Tened en cuenta que si trabajáis con Windows necesitaréis realizar un proceso algo diferente para poder disfrutar de Jekyll. Aquí os facilito otro enlace de otra página que explica como podéis hacerlo en windows: http://jekyll-windows.juthilo.com/

Espero que os haya resultado de interés y de ayuda. ¡Un saludo!
